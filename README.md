\# Classplus Lecture Logger 🎓



A lightweight Chrome Extension built to solve a common frustration: losing your place in online lectures. This extension automatically logs your watch history on Classplus web players and seamlessly auto-resumes videos right where you left off.



\## ✨ Features

\* \*\*Auto-Resume:\*\* Remembers exactly where you stopped watching a lecture and skips right to that timestamp the next time you open it.

\* \*\*Watch History:\*\* Keeps a running log of the last 10 lectures you watched, accessible instantly via the extension popup.

\* \*\*Smart Tracking:\*\* Uses a 3.7-second safety shield to prevent accidental 00:00 overwrites if you quickly click and close a video.

\* \*\*SPA Compatible:\*\* Built to handle React Single Page Application DOM changes and Video.js streaming quirks without rubber-banding.



\## 🚀 Installation (Developer Mode)

Since this extension is not currently on the Chrome Web Store, you can install it locally:



1\. Download or clone this repository: `git clone https://github.com/YOUR\_USERNAME/classplus-lecture-logger.git`

2\. Open your Chromium-based browser (Chrome, Edge, Brave) and navigate to the extensions page (`chrome://extensions/`).

3\. Toggle \*\*"Developer mode"\*\* on in the top right corner.

4\. Click \*\*"Load unpacked"\*\* and select the folder containing this repository.

5\. Pin the extension to your toolbar and head over to Classplus!
